# JARVUS AI Assistant

A powerful desktop AI assistant built using **Python + PyQt6** that combines desktop automation, productivity management, scientific calculations, system monitoring, and study tools into one intelligent interface.

## Features

### Desktop Automation

* Open desktop applications
* Close running applications
* Show desktop instantly
* View desktop files
* File finder across system directories
### Web Intelligence
* Google search integration
* YouTube creator search
* GitHub profile search
* Repository opener
* LinkedIn profile search
* GitHub trending explorer
### System Monitoring
* Battery percentage monitoring
* CPU usage tracking
* RAM status viewer
* Disk usage scanner
* WiFi network scanner
* Current WiFi connection status
### Scientific Calculator
Supports:
* Addition
* Subtraction
* Multiplication
* Division
* Logarithms
* Antilog
* Square root
* Factorial
* Trigonometric functions (sin, cos, tan)
### Geometry Engine
**Area Calculations**
* Circle
* Rectangle
* Square
* Triangle
**Volume Calculations**
* Cube
* Sphere
* Cylinder
* Cone
### Productivity Tools
* Notes system
* Task manager
* Command history
* Clipboard reader
* Screenshot capture
* Timer system
* Pomodoro study sessions
### AI Memory System
Stores:
* Notes
* Tasks
* Command history
* User interactions
via:
`jarvis_memory.json`
### Study Assistant Mode
Designed for focused academic workflows:
* Focus session management
* Quick calculations
* Formula assistance
* Productivity tracking
### Voice Assistant
Speech recognition support for hands-free commands.
